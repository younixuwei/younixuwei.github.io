#模块化规范

服务器端规范

  • CommonJS
  • 同步加载，同步执行

浏览器端规范

  • AMD
    • Asynchronous Module Definition
      • RepuireJS
      • 预加载，预执行(加载一个执行一个)
  • CMD
    • Common Module Definition
      • SeaJS
      • 预加载，懒执行(所有文件加载完毕后执行)

官方解决方式

ECMAScript 6 中的 module 功能

  • export命令用于规定模块的对外接口
  • import命令用于输入其他模块提供的功能。
